Copy YouTube Transcript

A minimalist Chrome extension that lets you copy or download the full transcript of any YouTube video you're watching – with a single click.

Perfect for:

  • Podcast listeners
  • Students and researchers
  • AI/ChatGPT users who want clean transcript input
  • Anyone who needs transcripts fast and clean

Features:

  • One-click full transcript copy
  • Download as clean .txt file
  • Option to remove timestamps
  • Option to include video title
  • Add custom prompt before transcript (e.g. for ChatGPT)
  • Works with Shorts too
Install on Chrome

Why I built this:

I often watch long-form YouTube videos—lectures, interviews, podcasts—and use ChatGPT to analyze or summarize the content. Copying transcripts manually from the YouTube transcript panel felt slow and frustrating.

This extension eliminates that friction. No clutter, no scrolling, no annoying UI. Just one click, and you're done.
